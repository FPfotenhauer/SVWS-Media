# 1. Einleitung

SVWS‑Media ist eine webbasierte Anwendung zur Verwaltung und Ausleihe von Medien in Schulen und Bildungseinrichtungen. Die Software unterstützt Lehrkräfte, Bibliothekspersonal und Administrator:innen beim Erfassen, Katalogisieren, Ausleihen und Berichten von physischen und digitalen Medien. Durch die Integration mit dem SVWS‑System ermöglicht SVWS‑Media eine nahtlose Synchronisation von Benutzerdaten und schulischen Stammdaten.

Dieses Benutzerhandbuch richtet sich an Anwender:innen mit unterschiedlichen Rollen — von Lehrkräften über Bibliothekspersonal bis hin zu Systemadministratoren — und erklärt die wichtigsten Funktionen, typischen Arbeitsabläufe sowie Hinweise zur Administration und Fehlerbehebung.

**Ziele dieses Dokuments:**
- Überblick über Funktionen und Einsatzmöglichkeiten von SVWS‑Media
- Schrittweise Anleitungen für tägliche Aufgaben (Anlegen von Medien, Ausleihe, Rückgabe)
- Konfigurations- und Synchronisationshinweise für die Anbindung an SVWS
- Tipps zur Datensicherung, Wartung und Fehlerbehebung



Am Anfang jedes Kapitels finden Sie eine kurze Beschreibung, gefolgt von Schritt-für-Schritt-Anleitungen und Screenshots, wo sinnvoll. Wenn Sie weitere Themen vermissen, melden Sie sich bitte beim Support oder fügen Sie Vorschläge im Projekt-Repository ein.

**Wichtige Hinweise:**
- **Zielgruppe:** SVWS‑Media richtet sich an Schulen, denen ein einfaches, leicht bedienbares Tool zur Verwaltung von Medien ausreicht.
- **ISBN‑Erkennung:** Eine automatische ISBN‑Erkennung soll das Anlegen von Schulbüchern und anderen gedruckten Medien deutlich erleichtern.
- **Installation:** Das Tool kann sowohl im internen Schulnetzwerk als auch auf einem externen Webspace betrieben werden.
- **Offline‑Synchronisation:** Es ist vorgesehen, eine Synchronisation mit dem SVWS‑Server auch in Szenarien ohne dauerhafte Internetverbindung zu unterstützen.
